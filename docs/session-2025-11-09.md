## 학습 포인트

### 이 튜토리얼에서 배우는 것
1. **실전 프로젝트**: 단순 예제가 아닌 완전한 게임 구현
2. **React 핵심 철학**: 컴포넌트, 단방향 흐름, 불변성
3. **점진적 학습**: 작은 단계로 나누어 체계적으로 학습
4. **비교 학습**: jQuery와 비교하여 React의 장점 이해

### jQuery 개발자가 어려워하는 부분 해결
1. **명령형 → 선언형**: "어떻게"에서 "무엇을"로 사고 전환
2. **DOM 직접 조작 → 상태 관리**: React가 자동으로 DOM 업데이트
3. **전역 상태 → 컴포넌트 상태**: 상태를 적절한 위치에 배치
4. **Mutation → 불변성**: 상태를 직접 수정하지 않는 이유 이해
