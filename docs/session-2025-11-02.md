## 주요 개념 요약

### Week 0에서 다룬 내용
- **const/let**: 변수 선언, 블록 스코프
- **화살표 함수**: () => {}, this 바인딩
- **템플릿 리터럴**: `${variable}` 문자열 보간
- **구조 분해**: { name, age } = user
- **스프레드**: [...array], {...object}
- **모듈**: import/export

### Week 1에서 다룬 내용
- **Node.js**: JavaScript 런타임 환경
- **npm**: 패키지 관리자
- **VS Code**: 코드 에디터 및 확장 프로그램
- **개발 환경**: Create React App 실습

### Week 2에서 다룬 내용
- **JSX**: JavaScript 안에서 HTML 작성
- **컴포넌트**: UI를 독립적이고 재사용 가능한 조각으로 분리
- **Props**: 부모 → 자식 데이터 전달 (읽기 전용)
- **State**: 컴포넌트 내부 상태 관리 (useState Hook)
- **이벤트 핸들링**: 사용자 상호작용 처리
